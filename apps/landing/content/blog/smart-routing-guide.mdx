## Что такое Smart Routing?

Smart Routing — это система автоматического выбора канала доставки на основе нескольких факторов:

- **Стоимость** — бесплатные каналы имеют приоритет
- **Доступность получателя** — есть ли у него Telegram, email
- **Тип сообщения** — OTP требует мгновенной доставки
- **История взаимодействий** — какие каналы работают лучше

## Как это работает

Рассмотрим пример отправки OTP-кода:

```typescript
await client.send({
    channel: "auto",
    recipient: "+998901234567",
    message: "Ваш код: 123456",
    priority: "high"
})
```

OpenNotify выполнит следующую логику:

1. Проверит, подключен ли получатель к вашему Telegram боту
2. Если да — отправит в Telegram (бесплатно, мгновенно)
3. Если нет — отправит SMS через выбранного провайдера

## Настройка правил маршрутизации

Вы можете настроить собственные правила через Dashboard или API:

```typescript
// Создание кастомного правила
await client.routing.createRule({
    name: "OTP через Telegram с fallback на SMS",
    conditions: {
        messageType: "otp",
        recipientHasTelegram: true
    },
    actions: [
        { channel: "telegram", priority: 1 },
        { channel: "sms", priority: 2, delay: "5s" }
    ]
})
```

## Реальные результаты

Наши клиенты видят следующие результаты после внедрения Smart Routing:

- **60-70% экономии** на транзакционных уведомлениях
- **99.5% delivery rate** благодаря автоматическому failover
- **Сокращение времени доставки** за счет параллельной отправки

## Лучшие практики

### 1. Подключите Telegram бота

Чем больше пользователей подключено к вашему боту, тем больше экономия:

```typescript
// Получите ссылку для подключения бота
const link = await client.telegram.getSubscriptionLink({
    userId: "user_123"
})
// Отправьте эту ссылку пользователю
```

### 2. Настройте приоритеты для разных типов сообщений

- **OTP/Auth**: Telegram → SMS (fallback через 5 секунд)
- **Транзакции**: Email → Push
- **Маркетинг**: Email → Telegram

### 3. Используйте аналитику

Регулярно проверяйте Dashboard, чтобы понимать:

- Какие каналы работают лучше
- Где можно ещё оптимизировать
- Сколько вы сэкономили
